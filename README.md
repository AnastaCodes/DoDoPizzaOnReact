# Dodo Pizza on React

**A pizza-ordering interface inspired by Dodo Pizza — product catalog, pizza configurator and cart flow. Built as a team project by two developers working through branches and pull requests.**

> 🚧 **Status: work in progress.** The catalog and pizza-configuration modal are functional; cart and checkout pages are under construction.

## Features

- Product catalog with category sections (pizza, snacks, drinks…)
- Pizza configurator modal: size, dough and extra ingredients
- Client-side routing (React Router) for cart / checkout / confirmation pages
- Image carousels with react-slick
- Component architecture: `pages` → `sections` → `components` → `ui`, each with scoped styles

## Tech stack

React 18 · Vite 5 · React Router 6 · Sass + CSS Modules · react-slick

## Run locally

```bash
git clone https://github.com/AnastaCodes/DoDoPizzaOnReact.git
cd DoDoPizzaOnReact
npm install
npm run dev
```
